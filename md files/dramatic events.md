# Dramatic events

Dramatic events make Sheogorad more interesting, but don't permanently change it; they "resolve." See [historical events.](historical%20events.md)

See drama as leeway for the simulation.

Without randomized but controlled events, not much would happen:
* NPCs would be able to think, walk and talk to others
* But nothing would "happen"

The world would be stuck in an uneventful state, which is much like how morrowind the game already behaves.

Without dramatic modifiers or events, sheogorad is limited to showing how NPCs act in a world that we already know.


## Example Dramas
An example of dramatic events (the llm can generate more:)
- Ash storms
- A ship arriving at seyda neen
- Vivec ruling a new law

## Event, Effect -> Resolve

Dramatic events have an immediate event, then an effect, then resolve or "cancel:"

* Vivec sets a new law, locals respond, then accept the new law
* Settlement locals report a strange fog, then the fog slowly settles
* Settlement residents have a rat infestation, then only talk about it afterwards 

Dramatic events aren't permanent; new laws imposed are temporary (and rat attacks too.)

Dramatic events are not historical events.
