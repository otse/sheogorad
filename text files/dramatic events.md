# Drama
for an ai simulation of vvardenfell to work, it may need dramatic events...

> dramatic events are the leeway that makes stuff happen in the sheogorad simulation

without randomized but controlled events, not much would happen:
* npcs would be able to think, walk and talk to others
* but nothing would "happen"

the world would be stuck in an uneventful state, which is much like how morrowind the game already behaves

without dramatic modifiers or events, sheogorad is limited to showing how npcs act in a world that we already know

an example of dramatic events (the llm should be able to think these up):
- ash storms
- a ship arriving at seyda neen
- vivec ruling a new law

dramatic or narrative events may linger for a while...
> so that npcs no longer react to the immediate event, but its effect
> for example after vivec set a new law, npcs first act shocked, then later complacent
